Project Title: HourFlow

HourFlow is a skills exchange platform based on a 'time bank' concept, where users exchange services using time as currency instead of money. The platform enables individuals to connect, offer, and receive services in their local community, fostering collaboration and reducing dependency on money.

Key Features

- Time as Currency: Users exchange services using time credits, not money.
- Mutual Exchange: Users can offer and receive services based on time spent.
- Live Location Tracking: Services are available based on proximity, making it easy to find help nearby.
- Service Quality Ratings: After each exchange, users can rate the quality of services.
- Partnerships with Local Businesses: Collaborations with local businesses to offer rewards and promote community engagement.
- Rewards System: Service providers can earn rewards for contributing to the community.
